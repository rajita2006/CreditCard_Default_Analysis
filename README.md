# Credit Card Default Prediction System

## Overview
This project predicts whether a customer will default on credit card payments using Machine Learning.

## Features
- Risk score (0–100)
- Default prediction
- Streamlit web app

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
python main_analysis.py
streamlit run app.py
